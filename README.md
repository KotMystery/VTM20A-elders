# VTM 20A Elder Character Creator

Веб-приложение для создания персонажей-Старейшин (Elders) для Vampire: The Masquerade 20th Anniversary Edition.

## Особенности

- ✅ Полная поддержка правил создания Старейшин
- ✅ Интерфейс на русском языке
- ✅ Отслеживание очков распределения
- ✅ Система Freebies и опыта
- ✅ Сохранение/загрузка персонажей
- ✅ Работает в браузере без установки
- ✅ Бесплатный хостинг на GitHub Pages

## Правила для Старейшин

### Характеристики
- **Атрибуты:** 9/7/5 (максимум 6 до Freebies)
- **Способности:** 18/12/8 (максимум 5 до Freebies)
- **Дисциплины:** 7 очков
- **Предыстории:** 3 очка (только: Поколение, Стадо, Ресурсы, Слуги)
- **Добродетели:** 5 очков

### Freebies
- Базовые: 15
- Обязательный недостаток котерии: +7
- Личные недостатки: до +9
- **Итого:** 22-31 очков

### Опыт
- Стартовый опыт: 113 XP

### Поколение
- Базовое поколение: 9-е
- Предыстория "Поколение": понижает вплоть до 4-го за каждое потраченное очко
- Недостаток "Разбавленная кровь": повышает вплоть до 14-го за каждое потраченное очко 

## Разработка

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для продакшена
npm run build
```

## Технологии

- Vite - быстрая сборка
- Tailwind CSS - стилизация
- Vanilla JavaScript - без фреймворков
- GitHub Pages - бесплатный хостинг
